=====================================
  نظام إدارة الجدول المدرسي
  School Timetable Management System
=====================================

--- طريقة التشغيل ---

1. فك الضغط عن الملف
2. فتح موجه الأوامر (Terminal/CMD)
3. الدخول إلى مجلد backend:
   cd backend

4. تثبيت المكتبات:
   npm install

5. تهيئة قاعدة البيانات:
   node seed.js

6. تشغيل الخادم:
   npm start

7. فتح المتصفح على:
   http://localhost:3000

--- بيانات الدخول ---
اسم المستخدم: admin
كلمة المرور:  admin123

--- مميزات النظام ---
- إدارة الطلاب والمعلمين والمواد والفصول
- إنشاء جدول تلقائي مع كشف التعارضات
- تعديل يدوي للجدول
- تصدير Excel منسق
- تصدير PDF للطباعة
- واجهة عربية كاملة (RTL)

--- ملاحظات ---
- المنفذ الافتراضي: 3000
- قاعدة البيانات: SQLite (backend/database/db.sqlite)
- لا يحتاج لتثبيت قاعدة بيانات خارجية
